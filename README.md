# wasmpack-starter

## Arch

- `logics`: pure rust logics
- `wasm`: logics binding
- `web`: vite app

## Develop

```
# install wasm-pack
# cargo install cargo-watch
$ pnpm dev
```

## Build

```
$ pnpm build
```

---

## Clone templates

```
$ cargo generate --git https://github.com/mizchi/wasmpack-starter.git --name my-project
$ cd my-project
```
